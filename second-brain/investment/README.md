## 핵심 요약

### 원칙 4줄
1. **"무엇을" < "얼만큼"** - 인간이 통제 가능한 것에 집중
2. **집착하지 않을 max ↔ 기대될 min** - 베팅 규모 공식
3. **재미있으면 위험, 지루하면 올바름** - 도파민 = 리스크 신호
4. **1회가 아닌 N회 시행의 누적** - 확률과 기댓값
