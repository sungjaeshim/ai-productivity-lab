# 🔄 성재님 패턴

관찰된 사고방식, 선호도, 행동 패턴.

---

## 커뮤니케이션
- **짧고 핵심적인 것** 선호
- 장황한 설명 ❌
- 핵심 찌르는 질문으로 마무리 ✅
- **"AI는 한국어 잘하는 외국인"** → 영어로 소통하면 더 효율

## 효율성
- 비효율 싫어함 → "24시간인데 왜 놀아?"
- **"시간 > 돈, 빠름 > 완벩"**
- 자동화, 시스템화 추구
- 수동 소득/패시브 인컴에 관심
- **진짜 수동 소득 = 첫 만들 때 열심히**

## 학습
- 구체적 예시/케이스 선호
- 직접 경험 중시 (콘텐츠도 경험 기반)
- **"배운 건 즉시 적용"** (에이전틱 코딩 강의 → 프로젝트 적용)

## 투자
- 미국 주식 (고성장 테크)
- NQ 선물
- 자동매매 개발 중 (Python)
- **피보나치 물타기 전략** (연 67% 기대)
- **고점 추격 금지** (SNDK 교훈)

## 작업 방식
- 큰 문제 분해 → 작은 작업으로
- 병렬 가능한 것 먼저 처리
- 테스트/검증 반드시 포함
- "이거 해줘" → 분해 → 병렬+순차 → 보고
- **최대 8개 병렬** (CPU 100% 경험 후 안정화)

## 의사결정 기준
1. **시간 > 돈** (시간이 더 중요)
2. **빠름 > 완벽** (속도 우선)
3. **리스크 감수 → 리턴 추구**
4. **60% 미만 확신 = 물어보기**

## 싫어하는 것
- 느린 진행
- 느린 응답
- 자비스 죽거나 해킹당하는 것
- **애널리스트 목표가 < 현재가** (위험 신호)

## 장기 목표
- **돈 많이 버는 것**
- **계속 성장하는 것**
