## 💎 띵언

> **"왜 이것을 하는가?"**

---

## 🎯 핵심 통찰

### 목적이 모든 것을 결정한다

```
"왜 이것을 하는가?" → 답이 다르면 다른 프로젝트
```

**적용:**
- 투자 vs 수입 → 목적 다름 → 별도 프로젝트
- 책 vs 블로그 → 목적 같음 → 같은 프로젝트

---

## 📊 실전 적용

### 프로젝트 구분

| 질문 | 답 | 결정 |
|------|------|------|
| "왜 주식에 투자하는가?" | 돈 불리기 | 투자 프로젝트 |
| "왜 블로그를 하는가?" | 돈 벌기 | 수입 프로젝트 |
| "왜 러닝을 하는가?" | 건강 | 건강 프로젝트 |

### 목적이 같으면 합치기

```
책 → 블로그 → 뉴스레터
"왜 하는가?" → 지식 전달
→ 같은 프로젝트
```

### 목적이 다르면 나누기

```
주식 (돈 불리기) vs 블로그 (돈 벌기)
→ 다른 프로젝트
```

---

## 💡 더 넓은 적용

### 의사결정

```
"왜 이것을 하는가?"
→ 모르면 → 하지 마
→ 명확하면 → 해
```

### 우선순위

```
"왜 이것을 먼저 해야 하는가?"
→ 답이 없으면 → 안 해도 됨
→ 답이 명확하면 → 우선
```

### 복잡한 상황

```
여러 옵션이 있을 때:
"각각 왜 하는가?"
→ 목적이 같은 것끼리 묶기
→ 목적이 다른 것은 분리
```
