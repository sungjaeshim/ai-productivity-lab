# Memory Tag Taxonomy (품질 우선)

## 규칙
- 태그는 2~4개만 사용
- `영역 + 주제 + 상태` 조합 우선
- 유사 태그 통일 (예: #ops, #operation -> #ops)

## 권장 태그셋
- 영역: #ac2 #second-brain #infra #trading #personal
- 주제: #memory-search #embedding #fallback #telegram #discord #pipeline
- 상태: #done #wip #blocked
- 유형: #decision #incident #checklist #experiment

## 예시
- AC2 퍼블릭 반영 이슈: #ac2 #incident #ops #done
- 임베딩 튜닝: #memory-search #embedding #experiment #wip
