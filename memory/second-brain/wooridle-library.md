## 📘 전문성 (Expertise)

### 전문성
**핵심 한 줄**: 전문가의 지식을 추출하고 훈련하는 방법론  
**주요 내용**:
- 전문성의 특징: 분야 특화, 패턴 인식, 빠른 실행, 강력한 기억력, 깊은 이해, 자기관찰 스킬
- 전문가는 문제를 이해하는데 더 많은 시간을 사용
- 전문가는 정제된 인식 능력과 차이점 구분 능력 보유
- DeliberatePerformance, ReflectivePractice, AppliedCognitiveTaskAnalysis로 전문성 계발 가능

**성재 적용**: 투자/러닝/코칭 전문성 계발 시 패턴 인식과 자기관찰에 집중. 문제 이해에 시간 투자.  
**출처**: wiki.wooridle.net/전문성

### DreyfusModel (드라이퍼스 모델)
**핵심 한 줄**: 전문성 습득 5단계 모델 (Novice → Expert)  
**주요 내용**:
- **1단계 Novice**: 상황 무관 규칙(recipes) 필요
- **2단계 Advanced Beginner**: 상황별 요소 인식 시작
- **3단계 Competent**: 중요 요소 선택, 책임감 느낌, 문제해결 가능
- **4단계 Proficient**: 직관적 상황 인식 + 분석적 행동 선택
- **5단계 Expert**: 의식적 사고 없이 직관적 행동

**변화 자각 7단계**: 
1. 행동 인식 없음 → 2. 며칠 후 생각남 → 3. 직후 인식 → 4. 하는 중 인식 → 5. 직전 인식 → 6. 빈 공간 느낌 → 7. 다른 선택 가능

**성재 적용**: 현재 수준 파악 → 다음 단계 목표 설정. 의식적 연습으로 직관 형성.  
**출처**: wiki.wooridle.net/DreyfusModel

### CollaborativeDevelopmentOfExpertise (CDE)
**핵심 한 줄**: OJT에서 전문가→훈련생 지식 전이 프로그램  
**주요 내용**:
- 6단계: 사전 브리핑 → 과제 훈련 → 회고 준비 → 1:1 회고 → 팀 회고 → Follow Through
- **질문 전략**:
  - Probe: 구체적 사례, 직관 활용 순간 파고들기
  - Help: 필요한 도움 구체화
  - New perspective: 새로운 시각으로 사건 재해석
- **피해야 할 행동**: 일반론, 쉽게 구할 수 있는 정보, 지루함

**성재 적용**: 멘토링/코칭 시 구체적 사례 중심 질문. "전문성 총동원 순간" 찾기.  
**출처**: wiki.wooridle.net/CollaborativeDevelopmentOfExpertise

## 🎯 코칭 (Coaching)

### CleanLanguage (클린 랭귀지)
**핵심 한 줄**: 메타포를 활용한 비개입적 질문 기법  
**주요 내용**:
- David Grove가 트라우마 치료에서 개발
- **9개 기본 질문**:
  - "X에 대해 또 뭐가 있을까요?"
  - "그것은 어떤 종류의 X인가요?"
  - "그것은 어디에 있나요?"
  - "그 다음에는 어떤 일이 일어나나요?"
  - "X 이전에는 무엇이 일어나나요?"
  - "X는 어디에서 오나요?"
  - "그것은 무엇같은가요?"
- 메타포 변형으로 변화 유도

**Elephant Whispering 5단계**:
1. Get positive (긍정적 표현)
2. Dream big (Lazy Jedi 질문)
3. Metaphor magic (은유에 이름 붙이기)
4. What needs to happen
5. All Set? (Can I? Will I?)

**성재 적용**: 코칭 시 상대 언어 그대로 사용. 메타포 탐색. 변화 질문.  
**출처**: wiki.wooridle.net/CleanLanguage

### The15CoachingProficiencies
**핵심 한 줄**: Thomas Leonard의 15가지 코칭 역량  
**주요 내용**:
1. **도전적 대화**: 말하지 않은 것 듣기, 당연한 질문하기
2. **고객 발견**: 은사/재능/가치/꿈 발견하도록 돕기
3. **위대함 끄집어내기**: 더 높은 기준 요구
4. **고객 즐기기**: 장단점 모두 즐김 → 신뢰 형성
5. **최선의 노력 확장**: 2배/10배 더 요구
6. **호기심으로 항해**: 발견 사업
7. **완전함 인식**: 모든 상황의 perfection 찾기
8. **가장 중요한 것 집중**: 긴급 vs 중요
9. **명료한 대화**: 편견/판단 제거
10. **발견 공유**: 직감/관찰 공유
11. **고객 챔피언**: 격려 이상의 지지
12. **새 영역 진입**: 새로운 개념/모델 제시
13. **진리 즐기기**: 진실 탐구
14. **후원 환경 설계**: 자동 지원 시스템
15. **인간애 존중**: 한계 인정

**성재 적용**: 코칭 세션마다 1-2개 역량 의식적 적용. 특히 4, 10, 14번 강화.  
**출처**: wiki.wooridle.net/The15CoachingProficiencies

### FiveMinuteCoach
**핵심 한 줄**: 5분 만에 가능한 CleanLanguage 코칭 기법  
**주요 내용**:
| 코칭 목적 | Clean Questions |
|---------|----------------|
| 결과 만들기 | 무슨 일이 일어났으면 좋겠나요? (긍정 표현) |
| 결과 알아내기 | 어떤 X인가요? / X에 관한 다른 것? |
| 최종 결과 선택 | 어떨 것 같으세요? / 은유 이름 붙이기 |
| 액션 플랜 | 어떤 일이 일어나야? / 또 다른 일? / 할 수 있나요? 의향 있나요? |
| 변화 동기부여 | X가 일어나면, 무슨 일이 일어날까요? |

**성재 적용**: 짧은 코칭 세션에 활용. 긍정 표현 유도부터 시작.  
**출처**: wiki.wooridle.net/FiveMinuteCoach

## 💡 변화관리 (Change Management)

### ChangeManagement
**핵심 한 줄**: 개인/조직 변화를 일으키고 지속시키는 과정  
**주요 내용**:
- 구체적 desired outcome 필요 (어렴풋한 desire로는 불가)
- **변화 관리 핵심**:
  - 작은 변화도 변화 ("예전보다 조금 나아진 거")
  - 천리길도 한 걸음부터
  - 다양한 도구 종합 사용
  - 시간 오래 걸림, 후퇴/퇴행 용인
  - '나'를 제외한 '저들' 변화시키기 ❌ → '우리'가 변화하기 ✅
  - 조르거나 닥달 금지
  - 경험하게 하기
  - 작은 시도로 경험 축적
  - 1:1 관계 활용

**변화의 여러 측면**: 비저너리 (감정) + 교사/코치 (지식/능력) + 설계자 (습관/넛지)

**성재 적용**: 습관 변화 시 작은 성공 경험 축적. 자기효능감 먼저. 1:1 관계 중시.  
**출처**: wiki.wooridle.net/ChangeManagement

### ADKARmodel
**핵심 한 줄**: 변화 5단계 모델 - Awareness → Desire → Knowledge → Ability → Reinforcement  
**주요 내용**:
- **A**wareness: 변화 필요성 자각
- **D**esire: 참여/지원 열망
- **K**nowledge: 변화 방법 지식
- **A**bility: 일상적 실행 능력
- **R**einforcement: 변화 유지 강화

**핵심 인사이트**:
- Awareness와 Desire가 critical
- 각 단계 순차 필요 (awareness 없이 knowledge 강조해도 무의미)
- 비즈니스 축 + 개인 축 동시 고려

**성재 적용**: 변화 시도 전 현재 단계 파악 → 다음 단계 집중 공략.  
**출처**: wiki.wooridle.net/ADKARmodel

### Nudge (넛지)
**핵심 한 줄**: 행동 선택을 설계하는 MINDSPACE 프레임워크  
**주요 내용**:
- **M**essenger: 누가 전달하는지 중요
- **I**ncentives: 손실 회피 같은 심리적 지름길
- **N**orms: 다른 사람 행동에 영향
- **D**efaults: 미리 정해진 선택지
- **S**alience: 자신에게 연관/이득 있는 것
- **P**riming: 무의식 신호(cue)
- **A**ffect: 정서적 연대
- **C**ommitment: 일관성 욕구
- **E**go: 더 나은 존재로 여겨지기

**성재 적용**: 습관 설계 시 Default 활용. Commitment (공개 선언). Priming (환경 cue).  
**출처**: wiki.wooridle.net/Nudge

### AppreciativeInquiry (긍정 탐색)
**핵심 한 줄**: 문제 해결 대신 강점 기반으로 조직 변화  
**주요 내용**:
- **철학**: '우리'가 '나'보다 현명 / 긍정 정서 / 강점 기반 자발 참여
- **5 Principles**:
  - Constructionist: 협력적 탐구
  - Simultaneity: 질문=변화
  - Poetic: 스스로 재해석
  - Anticipatory: 긍정 미래 공유
  - Positive: 긍정성 효과 극대화

- **4D Process**:
  1. **Discovery**: 과거~현재 최고 순간 찾기 (긍정 인터뷰)
  2. **Dream**: 이상적 미래 상상
  3. **Design**: 가치 담은 도발적 제안 작성
  4. **Destiny**: 실천 + 지속적 학습

**성재 적용**: 팀 회고 시 "잘한 것" 먼저 탐색 → 미래 설계. 강점 기반 목표 설정.  
**출처**: wiki.wooridle.net/AppreciativeInquiry

---

### OpenSpaceTechnology
**핵심 한 줄**: 열정과 책임 기반 자율 토론 컨퍼런스  
**주요 내용**:
- Harrison Owen 개발 (휴식시간 토론에서 착안)
- **4 Principles**:
  1. 오는 사람이 맞는 사람들
  2. 어떤 결과든 유일한 것
  3. 시작한 시간이 맞는 시간
  4. 끝나면 끝난 것

- **Law of Two Feet**: 가장 기여/배움 얻을 곳으로 이동하라
- **Be prepared to be surprised**: 혁신은 놀라움에서

**성재 적용**: 팀 오픈 세션 기획 시 활용. 자율성 최대화.  
**출처**: wiki.wooridle.net/OpenSpaceTechnology

### WorldCafe
**핵심 한 줄**: 테이블 호스트 고정 + 참여자 섞기 대화 기법  
**주요 내용**:
- 4~6명 소그룹 테이블
- 테이블 호스트 1명 고정
- 20~30분 토론 → 참여자 섞어 이동 → 반복 3~4차례
- 테이블마다 종이에 공동 기록
- 마지막에 호스트 발표

**효과**:
- 문제의식/해결책 공감대 전체 확산
- Big mouth 분산
- 저항 적은 액션 아이템 도출

**성재 적용**: 팀 워크숍에 활용. 과거 공유 짧게 → 새 논의 집중.  
**출처**: wiki.wooridle.net/WorldCafe

## 🧭 리더십 & 의사결정

### ToJuniorManager
**핵심 한 줄**: 팀장/리더에게 필요한 역량과 원칙  
**주요 내용**:
- **원칙**:
  - 나이 먹고 치사해지지 말자 (실수 떠넘기기 금지)
  - SystemDynamics 이해 + Delay effect 다루기 (천천히, 기다리며)
  - 문제해결 리더십 기르기
  - 사람이 문제 → 환경이 문제일 수 있음
  - 팀원은 "자신의 환경에서 최선" 전제
  - 다른 곳에서 더 행복할 수 있음 인정
  - 문제 제기 팀원 = 카나리아 (문제 예민 캐치)
  - 어른 뽑고 어른 대우 (보모 역할 금지)
  - 규칙 만들면 나도 따르기
  - 내가 못할 것 요구 금지
  - 먼저 그 나라와 의를 구하라 (공 전에 일)

**성재 적용**: 리더십 상황 시 환경 먼저 점검. 팀원 신뢰. 나부터 실천.  
**출처**: wiki.wooridle.net/ToJuniorManager

### ElementsOfInfluence (영향력의 요소)
**핵심 한 줄**: 설득/영향력 14가지 접근법 + 4가지 Dark Side  
**주요 내용**:
**Rational (이성적)**:
- Logical persuading (논리)
- Legitimizing (권위)
- Exchanging (협상)
- Stating (단언)

**Social (사회적)**:
- Socializing (친밀감)
- Appealing to Relationship (관계)
- Consulting (질문/참여)
- Alliance building (연합)

**Emotional (감정적)**:
- Appealing to values (가치/감정)
- Modeling (롤모델/멘토링)

**Dark Side (부정적)**:
- Avoiding (회피)
- Manipulating (조작)
- Intimidating (위협)
- Threatening (강요)

**성재 적용**: 상황별 영향력 전략 선택. Dark Side 자각하고 피하기.  
**출처**: wiki.wooridle.net/ElementsOfInfluence

## 📝 메타 지식

### AiGarden
**핵심 한 줄**: AI-인간 협업 지식 저작 철학과 프로토콜  
**주요 내용**:
- **AI-Human 협업 원칙**:
  - Small Steps (작은 변화→즉시 검증)
  - Transparent Reasoning (추론 과정 명확)
  - Human Final Say (인간 최종 판단)
  - Iterative Refinement (점진적 개선)

- **Garden Philosophy**: Christopher Alexander의 "살아있는 구조"
  - Structure-Preserving Baby Steps
  - Living Centers & Explicit Assumptions
  - Unfolding Wholeness Through Validation
  - Generate, Don't Fabricate

- **협업 스타일**: Concise / Insight / Practical / Dense / Dialogue

**성재 적용**: AI와 협업 시 작은 단계로 검증. 인간 판단 우선. 점진적 개선.  
**출처**: wiki.wooridle.net/AiGarden
