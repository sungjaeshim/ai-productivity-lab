## 🌱 3단계 로드맵

### Phase 1: Notion 템플릿 (1주일)

**목표**: 빠르게 판매 가능한 형태 만들기

**작업**:
1. [ ] Notion 템플릿 구조 설계
   - 씨앗 카드 템플릿
   - 성장 로그 템플릿
   - 수확 체크리스트

2. [ ] 템플릿 채우기
   - 성재님 실제 아이디어 10개
   - 예시/가이드 추가
   - 태그 시스템 적용

3. [ ] 판매 준비
   - 상세 페이지 작성
   - 가격 책정 ($29-$49)
   - Gumroad/Notion marketplace 등록

**성공 기준**:
- 템플릿 완성
- 지인 5명에게 무료 배포
- 피드백 3개 이상 수집

---

### Phase 2: 자동화 (2주일)

**목표**: 아이디어 정원 자동 관리

**작업**:
1. [ ] 자동 분류 스크립트
   ```python
   # scripts/idea_garden_auto_sort.py
   # 대화/작업에서 아이디어 자동 추출
   # seeds/에 자동 등록
   # 태그 자동 부여
   ```

2. [ ] 상태 추적 대시보드
   - 씨앗 → 성장 → 수확 자동 전환
   - 주간/월간 리포트
   - Canvas 기반 시각화

3. [ ] Heartbeat 연동
   - 매일 새로운 아이디어 체크
   - compost/에서 패턴 발견
   - harvested/에서 성공 사례 추출

**성공 기준**:
- 자동 분류 정확도 80%+
- Heartbeat마다 자동 업데이트
- 성재님 만족도

---

### Phase 3: Canvas 웹앱 (1달)

**목표**: iPad/iPhone에서 바로 접근

**기능**:
1. 드래그앤드롭 카드
2. 시각적 태그 필터
3. 성장 트래커 (진행 바)
4. Notion 양방향 동기화

**기술 스택**:
- Canvas (Clawdbot 내장)
- Flask/FastAPI (백엔드)
- Notion API (동기화)

**성공 기준**:
- iPad/iPhone에서 정상 작동
- Notion과 실시간 동기화
- 성재님 자주 사용
