# AppreciativeInquiry (긍정 탐색)

## 핵심 한 줄
문제 해결 대신 강점 기반으로 조직 변화

## 철학
- '우리'가 '나'보다 현명
- 긍정 정서
- 강점 기반 자발 참여

## 5 Principles
1. **Constructionist** — 협력적 탐구
2. **Simultaneity** — 질문=변화
3. **Poetic** — 스스로 재해석
4. **Anticipatory** — 긍정 미래 공유
5. **Positive** — 긍정성 효과 극대화

## 4D Process
| 단계 | 내용 |
|------|------|
| 1 Discovery | 과거~현재 최고 순간 찾기 (긍정 인터뷰) |
| 2 Dream | 이상적 미래 상상 |
| 3 Design | 가치 담은 도발적 제안 작성 |
| 4 Destiny | 실천 + 지속적 학습 |

## 성재 적용
- 팀 회고 시 "**잘한 것** 먼저 탐색" → 미래 설계
- 강점 기반 목표 설정
