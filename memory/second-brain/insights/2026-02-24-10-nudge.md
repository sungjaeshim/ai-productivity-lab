# Nudge (넛지)
> 출처: wiki.wooridle.net/Nudge
> 태그: #Nudge #MINDSPACE #행동설계

## 핵심 한 줄
행동 선택을 설계하는 MINDSPACE 프레임워크

## MINDSPACE 9요소

| 요소 | 의미 | 성재 시스템 적용 |
|------|------|----------------|
| **M**essenger | 누가 전달하는지 중요 | Jarvis가 직접 전달 |
| **I**ncentives | 손실 회피 같은 심리적 지름길 | 손실 프레이밍 가끔 활용 |
| **N**orms | 다른 사람 행동에 영향 | "BIT Runners 기록" 언급 |
| **D**efaults | 미리 정해진 선택지 | 기본값 = 좋은 선택으로 설계 |
| **S**alience | 자신에게 연관/이득 있는 것 | 성재 상황과 직접 연결 |
| **P**riming | 무의식 신호(cue) | 모닝 브리핑 1줄 의도 |
| **A**ffect | 정서적 연대 | 감성 터치 가끔 |
| **C**ommitment | 일관성 욕구 | 공개 선언 → second-brain 저장 |
| **E**go | 더 나은 존재로 여겨지기 | "성재님 투자 철학답게~" |

## 성재 적용
- 습관 설계 시 **Default 활용**
- **Commitment** (공개 선언)
- **Priming** (환경 cue)

## 시스템 적용 (cron 문구)
MINDSPACE 넛지 설계 → 모닝 브리핑, 저녁 회고 메시지 문구에 즉시 적용
