# SatirChangeModel
> 출처: wiki.wooridle.net
> 카테고리: 학습/조직 (25개)
> 태그: #학습/조직 (25개)

- **핵심**: VirginiaSatir 가 정의한 변화의 과정에 대한 ChangeModel 이다
- **주요 내용**:
  - (Late) Status Quo Chaos Integration Practice New Status Quo 변화하기 이전의 평형 상태가, 어떤 외적인, 내적인 이유에서 변화를 겪는다...
  - 변화의 초기에는 혼돈이 일어나는데, 이 혼돈 상태를 해결하기 위해서 이전의 평형 상태로 되돌아갈 수도 있고, 이후의 통합 상태로 진행할 수도 있다...
  - 하지만 이 통합은 익숙치 않아서, 계속된 연습을 통해서 정착되고, 새로운 평형 상태가 된다...
- **성재 적용**: 실제 상황에 적용해보세요.
