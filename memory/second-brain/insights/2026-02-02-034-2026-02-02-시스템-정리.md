# 2026-02-02 시스템 정리

## 불필요한 시스템 메시지 제거

### 문제
cron job 실행 후 다음 메시지들이 자동 출력됨:
1. "MACD 신호가 발생했습니다. 텔레그램으로 알림을 보내겠습니다."
2. "출력 없음 → 아무것도 하지 않음"
3. "텔레그램 알림 전송 완료! 요약: ..."

### 원인
1. `macd_alert.py`에 중복 코드 및 불필요한 출력
2. `macd_check.sh` 필터링 부족
3. `nq_position_check.sh` 데이터 없을 때 메시지 출력

### 해결

#### 1. macd_alert.py 수정
- 중복된 `save_state()` 제거
- 신호 없을 때 아무것도 출력하지 않음

#### 2. macd_check.sh 수정
```bash
# 필터링에 추가
|| [[ "$RESULT" == *"텔레그램 알림 전송 완료"* ]]
```

#### 3. nq_position_check.sh 수정
```bash
# 신호 없으면 조용히 종료
if [ -z "$PRICE" ]; then
    exit 0
fi
```

#### 4. Cron job 재설정
- 기존 job 삭제
- 새 job 생성 (systemEvent 방식)
- 간격: 15분

### 결과
이제 MACD 신호 있을 때만 텔레그램 알림 전송 🔔
