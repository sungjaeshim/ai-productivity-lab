## ADMX 전략 (2026-02-02 개발)

### 전략 개요
- **대상**: NQ 선물 (Nasdaq 100 E-mini)
- **타임프레임**: 15분봉
- **구성**: MACD + ADX + RSI 결합

### 발견한 것
1. **NQ 특성**: High Beta (일일 변동성 1.88%), 추세성 강함
2. **MACD**: 추세 전환 신호 (단독 사용 시 위신호 많음)
3. **ADX (14)**: 추세 강도 확인 (25 이상 = 유효한 추세)
4. **RSI (14)**: 과매수/과매도 필터 (70/30)

### 백테스터 상태
- **파일**: `scripts/backtest_admx_strategy.py`
- **기능**: NQ 15분봉 데이터 → 승률/수익률 분석
- **다음 단계**: 데이터 수집 → 백테스트 실행 → 승률 55% 달성

### 리스크 관리
- **손절**: -1.5%
- **익절**: +2.5%
- **포지션 사이징**: 계좌의 2% 위험
