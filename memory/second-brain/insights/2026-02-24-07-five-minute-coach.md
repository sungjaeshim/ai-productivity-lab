## 핵심 한 줄
5분 만에 가능한 CleanLanguage 코칭 기법

## 5단계 질문 프로토콜

| 목적 | Clean Questions | 예시 |
|------|----------------|------|
| 결과 만들기 | "무슨 일이 일어났으면 좋겠나요?" | 긍정 표현 유도 |
| 결과 알아내기 | "어떤 X인가요? / X에 관한 다른 것?" | 구체화 |
| 최종 결과 선택 | "어떨 것 같으세요? / 은유 이름 붙이기" | 선택 |
| 액션 플랜 | "어떤 일이 일어나야? / 또 다른 일?" | 계획 |
| 변화 동기부여 | "X가 일어나면, 무슨 일이 일어날까요?" | 영향 |

## 의지 확인
- "할 수 있어요?"
- "할 의향 있어요?"

## 핵심 원칙
1. **긍정 표현 유도부터** 시작
2. 은유 탐색으로 **자기 인식**
3. 액션 조건 구체화
4. 의지/가능성 확인

## 성재 적용
짧은 코칭 세션에 활용

## Jarvis 적용 (AGENTS.md)
고민/문제 상황 시 **답 주기 전에 이 5단계 먼저**:

1. "무슨 일이 일어났으면 좋겠어요?" (긍정 결과)
2. "어떤 X인가요?" (결과 구체화)
3. "그것은 무엇 같아요?" (은유 탐색 → 자기 인식)
4. "어떤 일이 일어나야 할까요?" (액션 조건)
5. "할 수 있어요? 할 의향 있어요?" (의지/가능성 확인)
