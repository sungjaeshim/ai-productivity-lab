# ElementsOfInfluence (영향력의 요소)
> 출처: wiki.wooridle.net/ElementsOfInfluence

## 핵심 한 줄
설득/영향력 14가지 접근법 + 4가지 Dark Side

## Rational (이성적)
| 전략 | 설명 |
|------|------|
| Logical persuading | 논리 |
| Legitimizing | 권위 |
| Exchanging | 협상 |
| Stating | 단언 |

## Social (사회적)
| 전략 | 설명 |
|------|------|
| Socializing | 친밀감 |
| Appealing to Relationship | 관계 |
| Consulting | 질문/참여 |
| Alliance building | 연합 |

## Emotional (감정적)
| 전략 | 설명 |
|------|------|
| Appealing to values | 가치/감정 |
| Modeling | 롤모델/멘토링 |

## Dark Side (부정적) ⚠️
| 전략 | 설명 |
|------|------|
| Avoiding | 회피 |
| Manipulating | 조작 |
| Intimidating | 위협 |
| Threatening | 강요 |

## 상황별 전략 선택

| 상황 | 추천 전략 |
|------|-----------|
| 제안서 | Logical + Values |
| 영업 | Socializing + Relationship + Consulting |
| 긴급 | Legitimizing + Stating |
| 협상 | Exchanging + Alliance |

## 성재 적용
- 상황별 영향력 전략 선택
- **Dark Side 자각하고 피하기**
