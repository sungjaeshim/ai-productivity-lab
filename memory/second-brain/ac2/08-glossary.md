## EOA (Essence of Agility)

> "애자일은 불확실성이 높은 상황의 효과적인 대응 전략"

### 4가지 원칙

**1. Avoid Big Loss (큰 손실 피하기)**
- 자잘한 손해는 감수, 큰 손실은 피하기
- Redundancy: 중복 허용
- Detect early: 문제 빠르게 감지
- Asymmetry: 비대칭성 확립

**2. Learn as You Go (진행하면서 배우기)**
- 진행 과정에서 배움을 얻고 더 좋은 방향으로
- Feedback & adapt: 피드백 받고 재조정
- Receiving new info: 새로운 정보 얻기
- Many projects, Sequential: 순차적 진행

**3. Achieve Critical Early (핵심을 먼저, 적은 노력으로)**
- 마지막에 영향 주는 결과를 미리 확인
- Piecemeal & center first: 가장 중요한 것 먼저
- Work with stakeholders: 이해관계자와 함께

**4. Be Flexible (유연하게 대처)**
- High ground for change: 변화에 유리한 지점
- Real option: 선택 옵션 만들어두기
- Diversify: 다양하게 대응
- Slack: 여유 확보

## CTA (Cognitive Task Analysis)

> 전문가의 인지적 작업 분석, 암묵지 추출

### What Experts Know (전문가가 아는 것)
- **Perceptual skills**: 남들은 좋다고 해도 "뭔가 있어" 감지
- **Mental models**: 큰 그림
- **Sense of typicality**: 하나 보면 유형 파악 (패턴 인식)
- **Routines/Job smarts**: 업무 요령

### What Experts Can Do (전문가가 하는 것)
- Mental simulations (과거/미래 시뮬레이션)
- Spot anomalies (이상 감지)
- Find leverage points (기회 포착)
- Manage uncertainty (불확실성 관리)
- Plan and replan (계획 수정)
- Assess complex situations (복잡한 상황 평가)
- Manage attention (주의 관리)

## 퍼포먼스 공식

```
Performance = Valid Tacit Knowledge 
            × Valid Explicit Knowledge 
            × Practice 
            × Habit 
            × Meta-cognition 
            × Social Capital 
            × Motivation 
            × Tools
```

### 핵심
- 1 이하 소수점이면 곱할수록 점수 낮아짐
- 0이면 전체가 0 → **모든 요소 중요**
- **Valid가 핵심** - 타당한 지식이어야

### 각 요소

| 요소 | 의미 |
|------|------|
| 암묵지 | 직접 경험으로 얻은 지식 (Worked Example) |
| 명목지 | 논문, 책, 강의 (Do it well을 배워야) |
| 수련 | Simple → Complex로 진행 |
| 습관 | 단기 보상 없어도 장기적으로 중요한 것 |
| 메타인지 | "이 길 아니네" 알아차리고 방향 전환 |
| 소셜 스킬 | Listening, Communication, Persuasion |
| 동기 | 흥미와 열정 유지 |
| 도구 | 인지적 도구 포함 |

## Influence Without Authority

> 명시적 권한 없이 영향력 행사하기

### 6가지 원천 모델

|  | 동기부여 | 능력 |
|--|---------|------|
| **개인** | 원하는 것에 동기 강화 | 한계를 뛰어넘게 도움 |
| **사회** | 동료 압력 동력화 | 다수의 힘 활용 |
| **구조** | 보상 설계, 책임 요구 | 환경 바꾸기 |

### 핵심 인사이트
- 1-2개 사용 = 일반인
- 여러 개 사용 = 인플루엔서 (효과 10배)
- 5명 중 3명이 하면 나머지도 따라함
- 뛰어난 1명이 하면 지식 파워로 따라감

### Vital Behaviors 파악법
1. Notice the obvious
2. 예외적으로 잘 된 경우 찾기
3. 작은 행동 찾기
4. 가설 검증
