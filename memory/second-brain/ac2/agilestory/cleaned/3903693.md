# 개발자와 디자이너 모집

**원문URL:** https://agilestory.blog/3903693
**날짜:** 2007년 11월 2일

---

온라인 학습 생태계 구인 공고입니다. 8월 공고에 이어 추가적으로 개발자 1명, 디자이너 1명을 더 뽑습니다.

대안 교육/학습에 관심이 있고 인간적인 생활을 해보고 싶은 IT인들의 많은 지원 기대합니다.
