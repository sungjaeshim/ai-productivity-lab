# 구글 G메일은 "여전히 작업 중"

**원문URL:** https://agilestory.blog/4868441
**날짜:** 2009년 4월 9일

---

구글 G메일을 쓴지 꽤 오래된 것 같습니다. 현재 G메일 사용량은 5GB에 육박합니다(특별히 파일 저장공간으로 쓰고 있지는 않습니다). 그동안 참 편하게 잘 쓴 것 같은데 요즘 들어 불만이 생기고 있습니다.
1주일 정도 된 것 같은데 G메일을 쓰다보면 화면 상단에 "Still working..."(여전히 작업 중)이라는 경고가 뜹니다. 그러면 작동이 멈춘 상태입니다. 아무 입력도 받지 않습니다.
거의 10분에 한 번은 이 경고가 뜨고 멈춰버리는 것 같습니다. 그래서 무척 불편하네요. 다른 분들은 어떤가요?
--김창준
