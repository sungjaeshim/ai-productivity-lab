# Visible Learning 스터디 모집

**원문URL:** https://agilestory.blog/5464858
**날짜:** 2011년 1월 10일

---

Visible Learning이라는 책(간단한 소개는 cbirk의 리뷰 참고) 스터디 모집합니다. 교육효과에 대한 800여개가 넘는 메타분석을 합하고 비교 분석했습니다. 실제 연구수자로는 수천개가 넘겠죠. 정말 어떻게 해야 학생이 공부를 잘 할 수 있는지에 대해 가장 포괄적이고 신뢰할만한 정보를 주는 책이라고 생각합니다.
스터디는 매주 수요일 오전 10시~12시까지 진행합니다. 돌아가면서 발제를 할 것 같고, 진행 과정 중에 직접 논문 찾아읽는 것도 하지 싶습니다. 좀 타이트하게 해서 단기간에 끝내는 걸 목표로 하고요. 장소는 방배동 애자일컨설팅 사무실(이수역에서 도보 5분 이내)입니다.
관심있는 분들은 메일로 연락주세요. 금주 수요일부터 만납니다. 첫 미팅 때는 책 없어도 됩니다.
