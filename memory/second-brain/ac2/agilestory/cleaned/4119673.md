# 테스트 주도 개발 재출간판

**원문URL:** https://agilestory.blog/4119673
**날짜:** 2008년 1월 29일

---

(이미지 출처는 강컴)
표지도 바뀌었습니다. 초록색이랑 빨간색으로 테스트 실패/성공 상태를 상징하고 있네요. 보색 배합이라 약간 어지러워 보이기는 하지만 예전보다는 더 생동감이 느껴져서 좋네요.
한동안 품절이어서 책을 구하지 못한 분들이 꽤 계시더군요. 그냥 재쇄를 찍은 것이 아니고 "재출간판 서문"을 추가하느라 시간이 걸려서 공백이 생겼습니다. 저와 켄트벡이 책 출간 후 얻은 중요한 교훈을 새로운 서문으로 추가했습니다. 원래는 좀 더 많은 내용을 담으려고 했는데 그려러니 책 한 권이 필요할 것 같더군요. 그래서 한가지 골자만 담으려고 했습니다.
다음은 최근 테스트 주도 개발을 접했던 분들의 소감인데 링크를 겁니다. 테스트 주도 개발을 고민하는 분들에게 참고가 될 것 입니다.
--김창준
