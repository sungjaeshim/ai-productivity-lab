# 국내 출간 예정인 애자일 관련 서적

**원문URL:** https://agilestory.blog/1400950
**날짜:** 2006년 2월 9일

---

- Extreme Programming Installed XP도입을 위한 실전 입문
- 테스트 주도 개발
- 실용주의 프로그래머
- 자바 툴을 이용한 eXtreme Programming
- XP로 접근하는 IT 프로젝트 관리와 구현
- 초보자를 위한 eXtreme 프로그래밍 - 리팩토링과 기민한 모델링
- 실용주의 프로그래머를 위한 단위 테스트 with JUnit
- 기타 실용주의 프로그래머 시리즈들
등이 있습니다. 여러 출판사에서 고루 나왔네요. 여기에서 테스트 주도 개발은 저와 강규영씨가 공역을 했고, 실용주의 프로그래머는 저와 정지호씨가 공역을 했지요. 유일하게 국내 집필 서적이 있는데, "XP로 접근하는 IT 프로젝트 관리와 구현"이라는 책입니다. 제 생각에는 XP를 짤막하게 정리해서 보기에는 좋지만, XP를 도입한 국내 사례가 제시되지 않고 구체적이고 실전적인 내용이 좀 부족하지 않나 하는 느낌입니다.
어쨌건 올해나 내년에는 국내 XP 사례를 반영한 책을 직접 한 번 써볼 생각입니다.
이 외에 출간 준비 중인 책들이 있는데,
- User Stories Applied
- Extreme Programming Explained, 2nd Edition
- Fit for Developing Software
등이 있습니다. USA는 LG전자 생산기술원에 계신 심우곤씨가 다른 분들과 함께 공역을 하시고 2월 경에 출간할 예정입니다. 이 책은 제가 추천사를 썼습니다. XPE2E는 3월에 출간될 예정인데 제가 감수를 보고 있습니다. FIT는 제가 직접 번역을 하고 있는데 올해 안에는 출간을 해야지요. --;
모두 인사이트 출판사에서 준비하고 있는 책들인데, 아마 다른 출판사에서도 많이 출간을 준비하고 있으리라 생각합니다.
--김창준
