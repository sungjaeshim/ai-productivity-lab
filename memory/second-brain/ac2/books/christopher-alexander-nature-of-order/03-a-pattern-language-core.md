## 1) 정밀 요약 번역
*A Pattern Language*는 253개 패턴을 통해 도시-건물-디테일을 연결하는 설계 언어를 제시한다. 각 패턴은 문제와 해결안을 함께 제시하며, 패턴들끼리 네트워크로 연결되어 규모가 달라도 재사용이 가능하다.

## 2) 핵심 정의 3줄
1. 패턴은 정답이 아니라 검증 가능한 가설 단위다.
2. 패턴 언어는 전문가 문서가 아니라 공동 설계 도구다.
3. 작은 단위 패턴의 연결이 큰 스케일 품질을 만든다.

## 3) 왜 중요한가
- AC2 지식 설계 시 “원칙 나열”보다 “문제-해결-연결” 형식으로 재사용성을 높일 수 있다.
