# 글쓰기 & 커뮤니케이션

## 표현적 글쓰기 (Expressive Writing)

> "감정을 들여다보고 글로 표현하는 방법"

### 3가지 세션
1. **덜 읽기**: 주의 돌리기
2. **표현적 글쓰기**: 감정 들여다보기
3. **야생 퍼실리테이션**: 집단으로 문제 해결

### 방법
1. 15-20분 연속 쓰기
2. 검열 없이 (맞춤법, 문법 무시)
3. 감정에 집중
4. 끝나면 버려도 됨

### 효과
- 스트레스 감소
- 감정 정리
- 자기 이해 심화

## 살리고 글쓰기 가이드

> "목적 중심의 효과적인 글쓰기"

### 핵심 원칙
1. **목적 먼저**: 이 글로 뭘 하고 싶은가?
2. **독자 중심**: 누가 읽는가? 그들이 원하는 건?
3. **구조화**: 핵심 → 근거 → 행동 요청

### 살리고(SALIGO) 공식
- **S**ituation: 상황 설명
- **A**ction: 내가 한 것
- **L**earning: 배운 것
- **I**mpact: 영향/결과
- **G**oal: 다음 목표
- **O**utput: 산출물

## ChatGPT 프롬프트 자가발전

> "ChatGPT가 스스로 프롬프트를 개선하게 만들기"

### 기본 프롬프트
"이 질문을 더 좋게 개선해줘"를 반복

### 개선이 안 될 때
- 맥락 부족 → 배경 정보 추가
- 목적 불명확 → 원하는 결과 명시
- 너무 추상적 → 구체적 예시 요청

### 고급 기법
1. 역할 부여: "너는 ~전문가야"
2. 단계적 요청: "먼저 ~하고, 그 다음 ~해"
3. 예시 제공: "이런 형식으로 답해줘: [예시]"

## Writer's Workshop 진행 순서

피드백을 효과적으로 주고받는 방법:

1. 글 읽기
2. 저자 환영
3. 저자가 핵심 부분 읽기
4. 요약
5. 긍정적 피드백
6. 개선 제안
7. 저자 질문
8. 저자에게 감사

**핵심**: 저자는 빠지고 적기만 함 (방어 없이 수용)
