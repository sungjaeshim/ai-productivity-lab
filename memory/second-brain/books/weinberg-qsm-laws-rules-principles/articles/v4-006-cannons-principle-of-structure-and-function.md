# V4-006 — Cannon’s Principle of Structure and Function

## Meta
- Volume: 4 (Anticipating Change)
- Original: **Cannon’s Principle of Structure and Function**
- Statement: "Structure and function are inseparably related." (p.71)
- KR 제목: **캐넌의 구조-기능 원리: 구조와 기능은 분리될 수 없다**
- Confidence: High
- Tags: #structure #function #system-design

## 1) 정밀 번역
구조와 기능은 분리할 수 없을 정도로 서로 연결되어 있다.

## 2) 핵심 정의 3줄
1. 조직/시스템 구조가 기능 성능을 규정한다.
2. 기능 문제를 구조 변경 없이 해결하려 하면 한계에 부딪힌다.
3. 설계 개선은 기능 요구와 구조 설계를 함께 다뤄야 한다.

## 3) 왜 중요한가
- 반복되는 성능/협업 문제의 근본 원인을 구조 관점에서 찾게 한다.

## 4) 실무 적용 체크리스트
- [ ] 기능 저하 이슈를 구조 문제와 함께 분석하는가?
- [ ] 책임/정보 흐름 구조가 기능 목표와 정렬되어 있는가?

## 5) 오해/한계
- 오해: "구조는 그대로 두고 기능만 개선 가능."
- 한계: 과도한 구조 개편은 전환 비용을 키우므로 단계적 접근 필요.
