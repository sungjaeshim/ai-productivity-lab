## 1) 정밀 번역
서로 다른 탐지 기술을 결합하면 더 향상된 탐지 기술이 만들어진다.

## 2) 핵심 정의 3줄
1. 단일 테스트 방식의 한계는 조합 전략으로 보완 가능하다.
2. 정적분석+단위+통합+탐색 테스트의 혼합이 효과적이다.
3. 테스트 설계는 도구 선택이 아니라 포트폴리오 설계다.

## 3) 왜 중요한가
- 한 가지 테스트에만 의존하면 특정 결함군이 누락된다.

## 4) 실무 적용 체크리스트
- [ ] 탐지 기법별 타겟 결함 유형을 정의했는가?
- [ ] 중복 탐지와 사각지대를 분석했는가?

## 5) 오해/한계
- 오해: “최고의 단일 도구 하나면 충분.”
- 한계: 과도한 조합은 운영비를 올리므로 균형 설계 필요.
