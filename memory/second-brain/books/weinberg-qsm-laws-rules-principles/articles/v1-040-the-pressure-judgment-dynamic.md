# V1-040 — The Pressure/Judgment Dynamic

## Meta
- Volume: 1 (Systems Thinking)
- Original: **The Pressure/Judgment Dynamic**
- Statement: "Pressure → conformity → misestimating → loss of control → more pressure." (p.255)
- KR 제목: **압박/판단 동학: 압박이 판단을 왜곡하고 통제 상실을 가속한다**
- Confidence: High
- Tags: #pressure #judgment #control-loop

## 1) 정밀 번역
압박은 동조를 낳고, 동조는 오판을 낳으며, 오판은 통제 상실로 이어지고, 통제 상실은 더 큰 압박을 낳는다.

## 2) 핵심 정의 3줄
1. 압박은 단기 동원 효과보다 판단 왜곡 비용이 더 크다.
2. 집단 동조는 리스크 신호를 숨긴다.
3. 압박 루프를 끊으려면 공개된 이견과 완충장치가 필요하다.

## 3) 왜 중요한가
- 고압 환경에서 일정/품질 동시 붕괴가 반복되는 구조를 설명한다.

## 4) 실무 적용 체크리스트

## 5) 오해/한계
- 오해: “압박을 더 주면 성과가 난다.”
- 한계: 적정 긴장감은 필요하지만 과압은 역효과가 크다.
