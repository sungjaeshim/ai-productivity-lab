# V2-023 — The Second Rule of Failure Prevention

## Meta
- Volume: 2 (First-Order Measurement)
- Original: **The Second Rule of Failure Prevention**
- Statement: "A loss of X dollars is always the responsibility of an executive whose financial responsibility exceeds X dollars." (p.159)
- KR 제목: **실패 예방 제2규칙: 손실 규모만큼의 책임은 상위 재무 책임선에 있다**
- Confidence: High
- Tags: #failure-prevention #accountability #financial-governance

## 1) 정밀 번역
X달러의 손실은, X달러 이상을 책임지는 임원의 책임이다.

## 2) 핵심 정의 3줄
1. 실패 예방은 책임선 명확화 없이는 작동하지 않는다.
2. 책임 없는 권한, 권한 없는 책임 모두 재발 위험을 키운다.
3. 규모 기반 책임 배분은 예방 체계의 최소 조건이다.

## 3) 왜 중요한가
- 문제를 개인 실수로만 끝내지 않고 제도 개선으로 연결한다.

## 4) 실무 적용 체크리스트
- [ ] 사고 보고서에 규모 기반 책임선이 명시되는가?
- [ ] 손실 규모별 예방 통제 수준이 정의돼 있는가?

## 5) 오해/한계
- 오해: "책임선만 정하면 예방된다."
- 한계: 책임선은 출발점이며 실행 통제·훈련·관측이 함께 필요.
