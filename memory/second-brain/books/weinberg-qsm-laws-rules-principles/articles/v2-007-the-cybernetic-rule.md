# V2-007 — The Cybernetic Rule

## Meta
- Volume: 2 (First-Order Measurement)
- Original: **The Cybernetic Rule**
- Statement: "Only when you know the state of the system can you make sensible interventions." (p.31 / p.48)
- KR 제목: **사이버네틱 규칙: 상태 가시성이 있어야 제어가 가능하다**
- Confidence: High
- Tags: #cybernetics #visibility #control

## 1) 정밀 번역
시스템의 상태를 알 때에만, 시스템 행동을 제어하기 위한 합리적 개입을 할 수 있다.
(다르게: 가시성 없이는 제어 불가. 보이지 않으면 조종할 수 없다.)

## 2) 핵심 정의 3줄
1. 제어의 전제는 상태 관측이다.
2. 가시성 없는 운영은 감으로 운전하는 것과 같다.
3. 모니터링·알림·피드백 루프가 제어 시스템의 핵심이다.

## 3) 왜 중요한가
- 장애 대응 품질은 기술력보다 상태 가시성 품질에 좌우된다.

## 4) 실무 적용 체크리스트
- [ ] 핵심 상태 지표(SLI/SLO, 큐, 지연, 오류율)를 실시간 관측하는가?
- [ ] 관측값 기반 자동/수동 제어 규칙이 있는가?

## 5) 오해/한계
- 오해: "로그 많으면 가시성 충분."
- 한계: 데이터량보다 의사결정 가능한 신호 설계가 중요.
