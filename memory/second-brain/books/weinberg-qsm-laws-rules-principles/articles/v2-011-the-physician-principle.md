# V2-011 — The Physician Principle

## Meta
- Volume: 2 (First-Order Measurement)
- Original: **The Physician Principle**
- Statement: "Every process is created by people, and thus can be changed by people." (p.59)
- KR 제목: **의사 원리: 사람이 만든 프로세스는 사람이 바꿀 수 있다**
- Confidence: High
- Tags: #process #change #agency

## 1) 정밀 번역
모든 프로세스는 사람에 의해 만들어졌고, 따라서 사람에 의해 바뀔 수 있다.

## 2) 핵심 정의 3줄
1. 프로세스는 자연법칙이 아니라 설계 산물이다.
2. 문제 프로세스는 개선 가능하며, 책임과 권한이 핵심이다.
3. "원래 이렇게 해요"는 설명일 수 있으나 정당화는 아니다.

## 3) 왜 중요한가
- 비효율을 숙명처럼 받아들이는 조직 문화에서 변화의 출발점을 제공한다.

## 4) 실무 적용 체크리스트
- [ ] 현재 프로세스의 설계 의도와 실제 효과를 분리해 점검하는가?
- [ ] 변경 권한자/영향자/저항 요인을 명시하는가?

## 5) 오해/한계
- 오해: "사람이 만든 거니까 쉽게 바꾼다."
- 한계: 문화·이해관계·규제 제약으로 변경 난이도는 높을 수 있다.
