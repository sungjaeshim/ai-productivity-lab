# V3-025 — The Stupid Model

## Meta
- Volume: 3 (Congruent Action)
- Original: **The Stupid Model**
- Statement: "Never attribute to maliciousness that which can otherwise be attributed to stupidity." (p.211)
- KR 제목: **어리석음 모델: 악의로 단정하기 전에 무지/실수 가능성을 먼저 보라**
- Confidence: High
- Tags: #stupid-model #attribution #hanlons-razor

## 1) 정밀 번역
달리 설명 가능한 경우라면, 악의를 원인으로 단정하지 마라(먼저 어리석음/실수를 고려하라).

## 2) 핵심 정의 3줄
1. 악의 단정은 과잉 대응을 부른다.
2. 많은 문제는 의도적 공격보다 무지·혼선·실수에서 발생한다.
3. 원인 귀인을 보수적으로 하면 대응 비용을 줄인다.

## 3) 왜 중요한가
- 감정적 과대 반응을 줄이고 실용적 개선 행동으로 전환시킨다.

## 4) 실무 적용 체크리스트
- [ ] 악의 가설 전에 실수/구조 원인 가설을 먼저 검토하는가?
- [ ] 재발 방지책이 사람 비난이 아닌 시스템 개선에 초점 맞추는가?

## 5) 오해/한계
- 오해: "모든 문제는 단순 실수다."
- 한계: 반복적/의도적 위협은 별도 보안·징계 체계가 필요.
