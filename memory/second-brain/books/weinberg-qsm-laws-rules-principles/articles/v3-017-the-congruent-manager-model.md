# V3-017 — The Congruent Manager Model

- Confidence: High

## 1) 정밀 번역
리더십은 모든 사람이 문제 해결에 창의적으로 기여할 수 있도록 환경을 만드는 능력이다.

## 2) 핵심 정의 3줄
1. 리더는 답을 독점하는 사람이 아니라 환경 설계자다.
2. 합치적 리더십은 목표·행동·보상을 일치시킨다.
3. 팀 성과는 개인 지시보다 환경 품질에 크게 좌우된다.

## 3) 왜 중요한가
- 지속 가능한 고성과 팀의 핵심은 사람보다 시스템적 환경 설계임을 보여준다.

## 4) 실무 적용 체크리스트
- [ ] 팀이 자율적으로 문제를 제기·해결할 구조가 있는가?
- [ ] 목표/평가/보상이 같은 방향으로 정렬되어 있는가?

## 5) 오해/한계
- 오해: "좋은 리더는 모든 해답을 제공한다."
- 한계: 환경 설계도 실행 규율과 함께 가야 효과적.
