# V1-021 — The Natural Software Dynamic

## Meta
- Volume: 1 (Systems Thinking)
- Original: **The Natural Software Dynamic**
- Statement: "Human brain capacity is fixed, but software complexity grows at least as fast as the square of program size." (p.135)
- KR 제목: **자연 소프트웨어 동학: 인간 인지 한계와 소프트웨어 복잡도의 비대칭**
- Confidence: High
- Tags: #software-complexity #cognitive-limit #scaling

## 1) 정밀 번역
인간의 뇌 용량은 대체로 고정되어 있지만, 소프트웨어 복잡도는 프로그램 크기의 제곱 이상으로 증가한다.

## 2) 핵심 정의 3줄
1. 코드가 커질수록 인간 이해능력과 시스템 복잡도 격차가 벌어진다.
2. 개인 역량 향상만으로는 복잡도 증가를 상쇄할 수 없다.
3. 구조화/모듈화/자동화가 인지 한계를 보완한다.

## 3) 왜 중요한가
- “사람 더 열심히”로는 복잡도 문제를 해결 못 한다.

## 4) 실무 적용 체크리스트

- [ ] 인지부하 완화 장치(문서/툴/테스트)가 있는가?

## 5) 오해/한계
- 오해: “고수 한 명이면 복잡도 문제 해결.”
- 한계: 도구화/구조화 없이는 개인 역량도 곧 한계 도달.
