# 🧠 제2의 뇌 (Second Brain)

성재님과의 대화에서 추출한 인사이트, 패턴, 의사결정 프레임워크를 정리하는 공간.

## 구조

```
second-brain/
├── README.md           # 이 파일
├── insights.md         # 핵심 인사이트 모음
├── decisions.md        # 의사결정 프레임워크
├── patterns.md         # 성재님 사고/행동 패턴
├── ideas.md            # 아이디어 씨앗들
├── meta.md             # 메타인지 관찰
└── ac2/                # AC2 지식베이스 (AgileStory 정리)
    ├── README.md
    └── agilestory/
```

## 원칙

1. **핵심만** - 장황하게 X, 날카롭게
2. **실행 가능하게** - "그래서 뭐?" 답할 수 있어야
3. **연결하기** - 새 인사이트는 기존 것과 연결
4. **주기적 리뷰** - 오래된 것 다시 보며 패턴 발견
