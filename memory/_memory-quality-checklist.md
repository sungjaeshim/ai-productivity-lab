# Memory Search 품질 체크리스트 (주간)

## 1) 데이터 품질
- [ ] 신규 메모리에 3줄 요약 포함
- [ ] 태그 2~4개 규칙 준수
- [ ] 제목이 검색어 친화적(고유명사/주제 포함)

## 2) 검색 품질
- [ ] 골든 질의 10개 결과 점검
- [ ] 상위 5개 결과 중 관련 문서 3개 이상 포함
- [ ] 오래된/중복 문서로 상위 오염 여부 확인

## 3) 운영 품질
- [ ] memory status에서 provider/model 확인
- [ ] chunk/files 증가 추세 확인
- [ ] fallback 발생 로그(ollama 실패 시 openai) 점검
