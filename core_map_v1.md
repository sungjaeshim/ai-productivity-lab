# 인간 삶 핵심 원리 지도 v1.0

## 목적
이 지도는 사용자가 시각화된 단어 지도를 보다가 모르는 단어 하나를 골라 AI에게 던지면, 그 단어를 출발점으로 개념·중요성·삶의 예시·연결 단어·다음 질문으로 사고가 확장되도록 돕는 탐색형 인터페이스다.

## 중심 단어 10개
1. 의미
2. 편향
3. 적응
4. 인센티브
5. 신뢰
6. 권력
7. 경로의존성
8. 피드백
9. 프레이밍
10. 불확실성

## 파생 단어
- 의미 → 목적 / 가치 / 서사
- 편향 → 확증편향 / 과잉확신 / 선택적 지각
- 적응 → 학습 / 환경적합성 / 생존전략
- 인센티브 → 보상 / 유인구조 / 도덕적 해이
- 신뢰 → 협력 / 예측가능성 / 사회자본
- 권력 → 영향력 / 제도 / 정당성
- 경로의존성 → 초기조건 / 누적 / 잠금효과
- 피드백 → 강화 / 균형 / 지연
- 프레이밍 → 해석 / 맥락 / 서술방식
- 불확실성 → 확률 / 오류 / 후회

## 핵심 연결선 10개
- 의미 ↔ 프레이밍
- 의미 ↔ 신뢰
- 편향 ↔ 프레이밍
- 편향 ↔ 불확실성
- 적응 ↔ 피드백
- 적응 ↔ 경로의존성
- 인센티브 ↔ 권력
- 신뢰 ↔ 권력
- 경로의존성 ↔ 피드백
- 불확실성 ↔ 신뢰

## 사용 방식
1. 지도를 본다.
2. 눈에 걸리는 단어 하나를 고른다.
3. 그 단어를 AI에게 던진다.
4. AI가 개념, 중요성, 삶의 예시, 연결 단어, 다음 질문으로 확장해준다.

## 시각화 원칙
- 첫 버전은 트리형 + 핵심 연결선 혼합형
- 중심 단어 10개가 먼저 보일 것
- 각 중심 단어는 파생 단어 3개를 가질 것
- 중심 단어 간 핵심 연결선 10개는 유지할 것
- 너무 복잡하지 않게, 탐색 욕구가 생기게 만들 것
