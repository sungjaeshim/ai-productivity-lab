## Usage

```bash
# Generate inventory to stdout
./scripts/system-inventory-scan.sh

# Write to file
./scripts/system-inventory-scan.sh --out data/system-inventory.json

# Dry run (preview without writing)
./scripts/system-inventory-scan.sh --dry-run
```

## Security

- Sensitive patterns filtered: `token`, `key`, `secret`, `password`, `credential`, `auth`, `private`
- Git remote URLs have tokens replaced with `***`
- Only localhost ports > 1024 are scanned
- No environment variables or config files are exposed

## Limitations

1. **No Docker support** - Only systemd and direct node processes detected
2. **Static tag mapping** - Project tags must be manually defined in script
3. **No change detection** - Each run overwrites previous inventory; no diff/history
