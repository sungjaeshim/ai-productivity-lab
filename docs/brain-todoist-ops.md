# Brain Todoist Operations Guide

> PCM (Personal Content Management) 운영 자동화 - Todoist 중심 작업 추적

## 개요

이 문서는 Telegram → Discord → Todoist 운영 루프의 Todoist 부분을 자동화하는 스크립트 사용법을 다룹니다.

## 구성요소

### 1. Stale Check (`brain-todoist-stale-check.sh`)

24시간 이상 상태 변화가 없는 작업을 탐지하여 BLOCKED 후보를 식별합니다.

```bash
# 기본 실행 (24h 임계값)
./scripts/brain-todoist-stale-check.sh

# 48시간 임계값
./scripts/brain-todoist-stale-check.sh --hours 48

# JSON 출력 (파싱용)
./scripts/brain-todoist-stale-check.sh --output json

# 드라이런 (API 호출 없이 미리보기)
./scripts/brain-todoist-stale-check.sh --dry-run
```

**출력 예시:**
```
=== Stale Task Report ===
Threshold: 24h (tasks not updated since 2026-03-01T09:30:00Z)

⚠️  Found 2 stale task(s):

[Active] 6g4CHMwJ... OpenClaw iOS 앱 Xcode 빌드 & 애플워치 연동 테스트
    Last update: 2026-02-22T00:54:14Z

[Waiting] 6g5RQjJj... 카카오 로그인 KOE004 처리
    Last update: 2026-02-27T09:26:22Z

📋 BLOCKED Candidates:
   Tasks above haven't moved in 24h. Consider:
   1. Move to Waiting if blocked externally
   2. Break down if too large
   3. Delete if no longer relevant
```

**종료 코드:**
| 코드 | 의미 |
|------|------|
| 0 | 성공 |
| 1 | API 에러 |
| 2 | 인증 에러 |
| 3 | jq 미설치 |

**한계점:**
- Todoist API가 전체 활동 히스토리를 제공하지 않음 (`updated_at`만 사용)
- 댓글 추가는 상태 변화로 감지되지 않음
- Rate limit: ~450 requests/15min

---

### 2. Done Summary (`brain-todoist-done-summary.sh`)

오늘 완료된 작업 중 상위 3개를 요약하여 Discord/Telegram용 메시지를 생성합니다.

```bash
# 기본 실행 (오늘)
./scripts/brain-todoist-done-summary.sh

# 특정 날짜
./scripts/brain-todoist-done-summary.sh --date 2026-02-28

# Markdown 출력
./scripts/brain-todoist-done-summary.sh --output markdown

# JSON 출력
./scripts/brain-todoist-done-summary.sh --output json

# 드라이런
./scripts/brain-todoist-done-summary.sh --dry-run
```

**출력 예시 (text):**
```
✅ Completed Today (2026-03-01): 3 task(s)

1. [Active] OpenClaw iOS 앱 Xcode 빌드 & 애플워치 연동 테스트
2. [Waiting] 카카오 로그인 KOE004 처리 완료
3. [Active] gog 빌드 - 릴리스 바이너리로 재시도
```

**출력 예시 (markdown):**
```markdown
## ✅ Completed Today: 3 task(s)

- [Active] OpenClaw iOS 앱 Xcode 빌드 & 애플워치 연동 테스트
- [Waiting] 카카오 로그인 KOE004 처리 완료
- [Active] gog 빌드 - 릴리스 바이너리로 재시도

---
_Generated by brain-todoist-done-summary.sh_
```

**종료 코드:** 동일 (stale-check와 동일)

**한계점:**
- `/tasks/completed` API가 최근 100개만 반환
- 프로젝트별 필터링은 클라이언트 사이드에서 수행
- KST 기준 날짜 계산 (서버가 Asia/Seoul TZ여야 정확)

## Cron 연동 예시

> **참고:** 실제 cron 등록은 하지 말고, 예시로만 참고하세요.

```bash
# 매일 오전 9시: 전날 완료 요약
0 9 * * * /root/.openclaw/workspace/scripts/brain-todoist-done-summary.sh --date yesterday

# 4시간마다: 정체 작업 탐지
0 */4 * * * /root/.openclaw/workspace/scripts/brain-todoist-stale-check.sh --hours 24

# 매주 월요일 오전 8시: 주간 정체 작업 리뷰 (48h 이상)
0 8 * * 1 /root/.openclaw/workspace/scripts/brain-todoist-stale-check.sh --hours 48
```

## 인증 설정

### 필수 파일

```
/root/.openclaw/credentials/
├── todoist              # Todoist API 토큰
└── todoist-projects.json # 프로젝트 ID 매핑
```

### todoist-projects.json 형식

```json
{
  "queue": "PROJECT_ID_1",
  "active": "PROJECT_ID_2",
  "waiting": "PROJECT_ID_3",
  "inbox": "PROJECT_ID_4"
}
```

### 토큰 획득

1. https://todoist.com/app/settings/integrations/developer
2. API token 복사
3. `credentials/todoist` 파일에 저장

## 문제 해결

### "jq is required" 에러

```bash
# Debian/Ubuntu
apt install jq

# macOS
brew install jq
```

### "Todoist token not found" 에러

```bash
# 토큰 파일 확인
cat /root/.openclaw/credentials/todoist

# 권한 확인
chmod 600 /root/.openclaw/credentials/todoist
```

### "API returned HTTP 403" 에러

- 토큰이 만료되었거나 잘못됨
- Todoist 설정에서 토큰 재발급 필요

### 빈 결과 (No stale tasks / No tasks completed)

- 정상적인 상황일 수 있음
- 날짜 범위 확인 (KST vs UTC)
- `--dry-run`으로 설정값 검증

## 운영 워크플로우 제안

1. **오전 (09:00)**
   - 전날 완료 요약 확인
   - 정체 작업 리뷰

2. **오후 (14:00, 18:00)**
   - 24h 정체 작업 체크
   - BLOCKED 후보 처리

3. **주간 (월요일)**
   - 48h+ 정체 작업 정리
   - Waiting 프로젝트 아이템 검토
